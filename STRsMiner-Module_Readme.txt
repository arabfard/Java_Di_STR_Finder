java -jar STRsMiner-Final_ALL.jar file.fa m1 M1 n1 N1


Description:
The values ​​of "m1" and "M1" correspond to the minimum and maximum values ​​of the STR core.
Also, The values ​​of "n1" and "N1" correspond to the minimum and maximum values ​​of the STR repeat.
The "file.fa" parameter is the same file with Fasta format, which we have prepared a separate file for each chromosome in this study.
